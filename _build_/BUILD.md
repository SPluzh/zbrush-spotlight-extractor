# Сборка zsltoimg в один EXE-файл в папке `_build_`

Для сборки консольного скрипта Python в единый независимый исполняемый `.exe` файл используется **PyInstaller**. В репозитории уже подготовлена вся необходимая структура в папке `_build_`.

> [!NOTE]
> Все сборочные скрипты настроены так, чтобы запускаться в Windows из папки `_build_` в один клик.

---

## 📋 План сборки

1. **Установка зависимостей**
   Убедитесь, что в вашей системе установлены Python 3, библиотека обработки изображений `Pillow` (зависимость экстрактора) и сам сборщик `PyInstaller`.
   ```bash
   pip install pyinstaller pillow
   ```

2. **Конфигурация сборщика (`_build_\zsltoimg.spec`)**
   Мы подготовили файл спецификации PyInstaller, который содержит точные настройки сборки:
   * **Один файл (`--onefile`)**: Все зависимости (Pillow, zslformat) упаковываются в один файл `zsltoimg.exe`.
   * **Консольный режим (`console=True`)**: Оставляет открытым окно консоли для вывода логов работы экстрактора и работы drag-and-drop.
   * **Поиск модулей (`pathex=['..']`)**: PyInstaller корректно находит модуль парсера `zslformat.py` в родительской директории.

3. **Скрипт автоматизации сборки (`_build_\__build.bat`)**
   Чтобы сборка проходила в один клик, создан Windows-скрипт `__build.bat`. Он выполняет следующие шаги:
   * **Проверка среды**: Проверяет наличие Python и PyInstaller в переменной окружения `PATH`.
   * **Очистка**: Удаляет следы прошлых сборок (`build/`, `dist/`, старый `zsltoimg.exe`), чтобы избежать конфликтов кэширования.
   * **Компиляция**: Запускает сборку через команду `python -m PyInstaller zsltoimg.spec`.
   * **Управление результатом**: Перемещает готовый `zsltoimg.exe` из временной папки `dist/` напрямую в папку `_build_`.
   * **Финальная чистка**: Полностью стирает временные папки `build/` и `dist/`, оставляя папку `_build_` чистой.
   * **Показ результата**: Автоматически открывает папку с готовым EXE в Проводнике Windows.

---

## 🚀 Инструкция по сборке

1. Перейдите в папку `_build_`.
2. Дважды кликните по файлу `__build.bat` (или запустите его из консоли).
3. Дождитесь надписи `BUILD SUCCESSFUL` и закрытия консоли сборщика.
4. В папке `_build_` появится готовый файл **`zsltoimg.exe`**.

> [!TIP]
> **Как использовать собранный EXE:**
> Вы можете просто перетащить любой `.zsl` файл (ZBrush Spotlight) на `zsltoimg.exe` с помощью мыши (Drag-and-Drop). Программа автоматически создаст папку `<имя_файла>_extracted` рядом с оригиналом и извлечет туда все текстуры в формате PNG.
